# SnapMind 记忆系统 v2.0.06 ⭐⭐⭐

> 三层精简架构 + 累积式项目记忆 + 自动归档

## 🏗️ 架构（方案C）

```
┌─────────────────────────────────────────────────┐
│  第1层：全局记忆 global-memory.md               │
│  所有项目状态 + 用户偏好                        │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│  第2层：项目记忆 projects/{项目}.md （累积式）  │
│  最近10条工作 + 关键决策（自动提取）            │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│  第3层：完整快照 snapshots/{项目}/              │
│  详细内容，按需加载                             │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│  第4层：归档 archive/{项目}/                    │
│  超30天自动归档，不参与日常加载                 │
└─────────────────────────────────────────────────┘
```

## 🛠️ MCP 工具

| 工具              | 功能                           |
| ----------------- | ------------------------------ |
| `save_snapshot`   | 保存 + 更新项目记忆 + 更新全局 |
| `load_memory`     | 全局 + 项目记忆 + 按需完整快照 |
| `get_status`      | 系统状态                       |
| `search_history`  | 搜索（含归档）                 |
| `check_health`    | 健康度检查                     |
| `merge_snapshots` | 合并快照                       |
| `export_memory`   | 导出 ZIP                       |
| `import_memory`   | 导入 ZIP                       |

## 📋 核心改进

### 1. 项目记忆（累积式）
- 每个项目一个文件 `projects/{项目}.md`
- 保留最近 10 条工作记录
- 自动从 tech-decisions.md 提取关键决策
- 不再每次覆盖，而是追加

### 2. 删除冗余层
- ❌ 删除精炼快照 (digest)
- ❌ 删除记忆快照 (memory)
- ✅ 只保留完整快照

### 3. 自动归档
- 超过 30 天的快照自动移到 archive/
- 搜索时可以搜到归档内容
- 不参与日常加载，节省 token

## 📊 加载策略

```
新会话默认加载：
- 全局记忆 (~300 tokens)
- 所有项目记忆 (~500 tokens/项目)
- 最新1条完整快照 (~800 tokens)

按需加载：
- 用户提到项目时，加载该项目最新快照
```

## 🗂️ 目录结构

```
.kiro/
├── memory/
│   ├── global/global-memory.md    # 全局记忆
│   ├── projects/{项目}.md         # 项目记忆（新）
│   ├── archive/{项目}/            # 归档
│   └── tech-decisions.md          # 技术决策
├── snapshots/{项目}/              # 完整快照
└── task-snapshot-index.txt        # 序号
```
