# SnapMind 分发包 v2.0.06

> 三层精简架构 + 累积式项目记忆 + 自动归档

## 🏗️ 新架构

```
第1层：全局记忆 → 所有项目状态
第2层：项目记忆 → 累积式，最近10条+关键决策
第3层：完整快照 → 详细内容，按需加载
第4层：归档     → 超30天自动归档
```

## 📋 v2.0.06 改进

- ✅ 项目记忆累积式更新（不再覆盖）
- ✅ 自动从 tech-decisions 提取关键决策
- ✅ 删除冗余的精炼/记忆快照
- ✅ 超30天快照自动归档
- ✅ 搜索支持归档内容

## 🚀 快速安装

```powershell
.\install.ps1
```

## 🛠️ MCP 工具

| 工具              | 功能                |
| ----------------- | ------------------- |
| `save_snapshot`   | 保存 + 更新项目记忆 |
| `load_memory`     | 加载全局+项目记忆   |
| `search_history`  | 搜索（含归档）      |
| `check_health`    | 健康度检查          |
| `merge_snapshots` | 合并快照            |
| `export_memory`   | 导出 ZIP            |
| `import_memory`   | 导入 ZIP            |
