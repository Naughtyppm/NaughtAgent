# 实现计划：Token 优化与命令增强

## 概述

本实现计划将 Token 优化和命令增强功能分为两个阶段实现。Phase A 专注于 Token 优化（降低消耗、支持长对话），Phase B 专注于命令增强（别名、历史、管道、链式执行）。

**重要说明**：部分功能已有基础实现，本计划仅扩展缺失功能：
- `src/context/index-cache.ts` - 已有基础缓存，需添加增量更新和统计
- `src/context/context-injector.ts` - 已有基础注入，需添加文件选择和会话摘要
- `src/command/builtin/history.ts` - 已有内存历史，需添加持久化

实现语言：TypeScript
测试框架：vitest + fast-check
