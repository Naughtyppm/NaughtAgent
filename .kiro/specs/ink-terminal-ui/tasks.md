# 实现计划: Ink 终端 UI

## 概述

使用 Ink (React for CLI) 重写 NaughtyAgent 的终端 UI，实现类似 Claude Code 的交互式体验。采用渐进式实现策略，先搭建基础框架，再逐步添加功能组件。
