# 产品概述

NaughtyAgent 是一个类似 Claude Code 的 AI 编程助手，采用 VS Code 扩展 + 独立后端服务的架构实现。

## 核心原则

1. **显式触发** - AI 仅在用户主动调用时工作，无后台进程
2. **人工确认** - AI 的修改必须经用户确认后才应用
3. **可控可维护** - 完全掌控代码，支持长期迭代

## 主要功能

- 代码阅读与分析
- 代码编辑与生成
- 文件操作（读/写/搜索）
- 命令执行
- 上下文管理
- 多轮对话

## 架构设计

分离式架构，包含两个主要组件：
- **VS Code 扩展**：UI 层，负责上下文收集、结果展示、用户交互
- **Agent 服务**：核心逻辑层，提供 CLI/HTTP 接口，调用 Claude API

## 权限模式（参考 Claude Code）

- **默认模式（ask）**：read-only，每次操作询问用户
- **Auto-accept 模式（allow）**：自动执行，不询问
- **Sandbox 模式**：沙箱内自由执行，超出边界才询问

Agent 根据任务自动选择需要的工具，权限通过配置控制，无需手动切换角色。
