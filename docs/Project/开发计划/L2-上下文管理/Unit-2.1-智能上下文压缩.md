## 目标

当上下文接近窗口限制时，智能压缩旧消息。

## 任务清单

- [ ] 实现消息重要性评分算法
- [ ] 实现摘要生成（调用 LLM 生成摘要）
- [ ] 实现渐进式压缩策略
- [ ] 保留关键信息（工具调用结果、用户指令）
- [ ] 添加压缩触发阈值配置

## 完成标准

- 上下文超过 80% 时自动触发压缩
- 压缩后保留关键上下文
- 用户无感知（除非查看历史）
