## 遇到的问题和解决方案

### 1. TypeScript 继承 Error 的原型链问题
**解决方案**：使用 `Object.setPrototypeOf()` 保持正确的原型链

### 2. 会话分支的数据隔离
**解决方案**：使用 `Array.slice()` 创建浅拷贝

### 3. 成本统计的平均值计算
**解决方案**：正确处理除零情况，返回 0

### 4. 数据迁移的安全性
**解决方案**：默认创建备份，支持强制迁移
