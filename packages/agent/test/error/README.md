## 注意事项

### Unhandled Rejection 警告
测试运行时会出现一些 "Unhandled Rejection" 警告，这是预期的行为。这些警告来自于我们故意抛出错误来测试错误处理逻辑。所有测试都正确地捕获和验证了这些错误。

### 使用 Fake Timers
重试测试使用 vitest 的 fake timers 来模拟时间流逝，避免测试中的实际延迟。这使得测试运行更快，同时仍然能够验证指数退避的正确性。
