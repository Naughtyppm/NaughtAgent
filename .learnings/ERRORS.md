# 错误记录

> 记录重复出现的错误，用于创建 troubleshooting Skill

---

## 记录格式

```markdown
### ERR-YYYYMMDD-NNN

- **Pattern-Key:** `error.category.specific`
- **Recurrence-Count:** N
- **First-Seen:** YYYY-MM-DD
- **Last-Seen:** YYYY-MM-DD
- **Status:** recorded | skill-created

**错误描述：**
错误信息和触发场景

**解决方案：**
如何修复

**相关 Skill：**
- Skills/xxx-troubleshooting/SKILL.md（如已创建）
```
